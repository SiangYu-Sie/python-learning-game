Python 10 等級學習內容與關卡規劃

一步一步引導機制 (Step-by-step Guidance)

每個等級的關卡將細分為三個引導階段：
1. 觀察與提示：使用 Guided Highlights (引導高亮) 標記編輯器，顯示操作說明，如：「請在此處輸入 x = 10 儲存你的魔力」。
2. 嘗試與執行：點擊執行，Pyodide 在前端沙盒執行後，顯示成功充能的角色動畫。
3. 實戰挑戰：要求使用者寫出能對怪物打出致命一擊的 Python 語句，正確後 Level Up (升級) 並解鎖下一關。

Lv.1

勇者覺醒

print() 與文字輸出

引導學習 print() 函數，在終端機輸出勇者名字，發送第一個覺醒信號。

Lv.2

魔法口袋

變數與基本資料型態

認識變數概念與整數、浮點數、字串型態，將魔力值儲存於口袋變數中。

Lv.3

煉金工坊

算術運算與字串操作

學習加減乘除與字串相加，用代碼計算物理傷害或合成魔法藥水。

Lv.4

分支岔路

if - elif - else 條件分支

根據怪物屬性，利用 If-else (條件分支) 判斷式，選擇對應魔法發動攻擊或防守。

Lv.5

冒險指南

邏輯運算與比較運算

使用 and, or, not 以及比較運算子，破解複雜的寶箱陷阱與結界挑戰。

Lv.6

聚能魔法陣

while 條件迴圈

使用 While Loop (While 迴圈)，在自身 HP 大於特定比例時持續聚能，施展大絕招。

Lv.7

分身斬

for 迭代迴圈與 range()

學習 For Loop (For 迴圈) 與 range() 函數，發動分身連續劍技一次擊倒複數敵人。

Lv.8

卷軸收納盒

List (列表) 基礎操作

學習 List (列表) 容器，用於儲存背包中的裝備、藥水，進行增刪查改操作。

Lv.9

終極契約

自訂 Function (函式) 定義

定義並命名自己的魔法招式，帶入不同的魔法參數 (Arguments)，封裝重複的技能邏輯。

Lv.10

封魔戰役

Return (傳回值) 與終極對決

學習 Return (傳回值) 機制，使自訂函式傳回傷害數值，擊敗深淵大魔王完成冒險。

開發關卡狀態機，管理 1-10 個 Level 狀態。設計多步驟 (Step-by-step) 引導狀態引擎，能夠讓每個關卡劃分為「提示、嘗試、實戰挑戰」三個連續進度，並能利用 Local Storage (本地儲存空間) 保存當前進度。

4.5 小時

T3: 程式碼編輯器與 Guided Highlights (引導高亮) 元件實作 Priority: Medium

實作程式碼編輯介面，加入行號標記。開發引導提示氣泡 (Tooltip) 與高亮元件，能配合關卡進度在程式碼特定行或特定輸入位置顯示閃爍引導框，協助初學者「一步步」跟隨操作。
